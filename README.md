# tactical-rpg-boardgamesaga

A board game adaptation of "The Saga" tactical RPG.

## Core Design Philosophy
1. **3d6 Cascading Resolution**: A systematic 3-stage roll (Skill, Result, Consequence) to simulate complex physics with deterministic flair.
2. **Neat & Consistent**: Modular rulebook structure to minimize mental load and maximize adaptability.
3. **Interactive Bookkeeping**: Designed for use with interactive character sheets to handle granular state-tracking (Shock, Bleeding, Balance).
4. **Battle-First Architecture**: Developing the tactical depth of combat as the foundation before expanding into campaign and world layers.
